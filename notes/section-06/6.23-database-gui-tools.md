# 6.23 Database GUI Tools - Study Notes

## Overview

Database GUI tools like phpMyAdmin, Adminer, and Sequel Pro provide visual interfaces for safely inspecting and editing WordPress data, browsing tables, querying data, and visualizing table structures.

**Key Reference:** [phpMyAdmin](https://www.phpmyadmin.net/) | [Adminer](https://www.adminer.org/) | [Sequel Pro](https://www.sequelpro.com/)

---

## 1. Browsing and Querying WordPress Tables

### phpMyAdmin

**Access:**
- Usually at `/phpmyadmin`
- Login with database credentials
- Browse WordPress tables

**Features:**
- Visual table browser
- SQL query interface
- Export/import data
- Table structure view

### Adminer

**Access:**
- Single PHP file
- Lightweight alternative
- Similar features to phpMyAdmin

### Common Operations

**Browse Tables:**
- Click table name
- View all rows
- Filter and search
- Sort columns

**Run Queries:**
```sql
SELECT * FROM wp_posts WHERE post_status = 'publish';
SELECT COUNT(*) FROM wp_users;
```

---

## 2. Safely Updating and Exporting Data

### Safe Updates

**Best Practices:**
- Always backup first
- Test queries in staging
- Use transactions when possible
- Verify changes

**Example Updates:**
```sql
-- Update option
UPDATE wp_options
SET option_value = 'new_value'
WHERE option_name = 'siteurl';

-- Update post
UPDATE wp_posts
SET post_title = 'New Title'
WHERE ID = 123;
```

### Exporting Data

**Export Options:**
- Export entire database
- Export specific tables
- Export as SQL
- Export as CSV

**Export via GUI:**
- Select database/table
- Click Export
- Choose format
- Download file

**View Relationships:**
