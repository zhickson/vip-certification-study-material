# 6.25 Site Health and Site Info - Study Notes

## Overview

WordPress Site Health provides built-in diagnostics to review site status, inspect environment details, and identify misconfigurations or missing modules.

**Key Reference:** [Site Health Screen - WordPress Codex](https://wordpress.org/support/article/site-health-screen/)

---

- Or `/wp-admin/site-health.php`

### Health Status

**Information Displayed:**
- Overall health score
- Critical issues
- Recommended improvements
- Performance checks

### Common Issues

**Typical Problems:**
- PHP version outdated
- Missing PHP extensions
- Database issues
- Security recommendations

---

## 2. Inspecting Environment Details via Site Info

### Site Info Tab

**Information Displayed:**
- WordPress version
- PHP version
- Database version
- Server information
- Active plugins/themes
- Configuration details

### Export Info

**Copy Site Info:**
- Click "Copy site info to clipboard"
- Paste for support
- Share with developers

---

## 3. Identifying Misconfigurations or Missing Modules

### PHP Extensions

**Check Extensions:**
- Required extensions listed
- Missing extensions highlighted
- Recommendations provided

### Configuration Issues

**Common Problems:**
- Memory limits
- Execution time limits
- File upload limits
- Database connection

## Exam Tips

### Key Points to Remember

1. **Site Health:**
   - Overall status
   - Critical issues
   - Recommendations

2. **Site Info:**

3. **Troubleshooting:**
   - Identify issues
   - Check recommendations
   - Fix misconfigurations
